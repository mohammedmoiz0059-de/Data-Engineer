### Source 
	Source system is the origin of data engineering lifecycle. 
	A source can be application message queue or transactional database.
	key things to consider:
		1. is it application data or data from Iot
		2. data is permanent or deleted quickly?
		3. rate at which the data is generated?
		4. Will it contains duplicates?
		5. Frequency at which the data should be pulled?
		6. if schema changes how it is communicated?
