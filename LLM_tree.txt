Large Language Models (LLMs) ⭐⭐⭐⭐⭐
- Transformer models trained on huge text/code
│
├── Closed / Commercial Models
│
│   ├── GPT
│   │   - OpenAI model family
│   │
│   │   └── ChatGPT
│   │       - Application built using GPT
│   │
│   ├── Claude
│   │   - Anthropic LLM family
│   │
│   └── Gemini
│       - Google LLM family
│
│
└── Open / Open-weight Models
    │
    ├── Llama
    │   - Meta open LLM family
    │
    ├── Mistral
    │   - Efficient open LLM family
    │
    └── Other open models
        - Community models


LLM Platforms / Model Hubs ⭐⭐⭐
- Places to access and manage models
│
├── Hugging Face ⭐⭐⭐
│   - AI model ecosystem/platform
│
│   Provides:
│
│   ├── Model Hub
│   │   - Download/use models
│   │
│   │   Examples:
│   │   - Llama models
│   │   - Mistral models
│   │   - Embedding models
│   │
│   ├── Transformers Library
│   │   - Python library to run models
│   │
│   ├── Datasets
│   │   - Training/evaluation datasets
│   │
│   └── Inference API
│       - Call models using API
│
│
└── Ollama ⭐⭐⭐
    - Run LLMs locally on your machine
    │
    Provides:
    │
    ├── Download models
    │
    ├── Run models offline
    │
    └── Local API endpoint
    │
    Examples:
    - Run Llama locally
    - Run Mistral locally

- Build applications using LLMs
│

│   Helps with:
│   - Prompt templates
│   - Chains
│   - Agents
│   - RAG pipelines
│
├── LlamaIndex

│
│   Helps with:
│   - Document loading
│   - Indexing
│   - Retrieval
│
└── Semantic Kernel
    - Enterprise AI orchestration
