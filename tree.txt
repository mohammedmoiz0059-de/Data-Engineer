└── Machine Learning (ML)
    - Systems learn patterns from data

    │
    ├── Learning Approaches (HOW models learn)
    │
    │
    ├── Supervised Learning
    │   │   - Learns from labelled data
    │   │
    │   ├── Regression
    │   │   - Predict numeric values
    │   │
    │   └── Classification
    │       - Predict categories/classes
    │
    ├── Unsupervised Learning
    │   - Learns without labelled answers
    │
    │   ├── Clustering
    │   │   - Groups similar data
    │   │
    │   └── Dimensionality Reduction
    │       - Reduces unnecessary features
    │
    ├── Semi-Supervised Learning
    │   - Mix of labelled and unlabelled data
    │
    ├── Self-Supervised Learning ⭐⭐⭐
    │   - Creates learning signals from data itself
    │
    │   └── Used heavily for:
    │
    │       LLM Pre-training
    │       - Predicting tokens from huge text
    │
    └── Reinforcement Learning
        - Learns through rewards
        │
        └── RLHF
            - Human feedback improves LLM behaviour
